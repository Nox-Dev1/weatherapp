### **_Welcome to weatherapp!_**

i made this app becuz in msn weather app there was sliding text showing weather and i said "why dont i just make it in scratch?"

This was completely made in Turbowarp (scratch fork)

Extensions used:

- Text extension by CST1229, BludIsAnLemon and Man-o-Valor: for turning spaces into + to be compatible with HTTP protocol

- Custom Styles by TheShovel: for modifying the ask ui

- JSON by Skyhigh173 and Mio: for the api

- Fetch: for GET requests

- Animated text by LilyMakesThings: for displaying the whole ui

The API i used for this is **WeatherApi.com**
